# Images

This directory holds documentation images for AV Morning Star.

## Contents

No images are currently committed to this directory. The `docs/images/` folder is reserved for application screenshots and UI illustrations.

To add the app screenshot referenced in the main README:

1. Take a screenshot of the running application.
2. Save it as `app-screenshot.png` in this directory.
3. The README already references it at `docs/images/app-screenshot.png`.
